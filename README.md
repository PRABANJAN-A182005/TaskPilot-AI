# 🚀 TaskPilot AI

> AI-Powered Project Execution & Collaborative Team Workspace

TaskPilot AI is a full-stack AI-driven project execution platform that evolved from a single-user task manager into a comprehensive collaborative team workspace with advanced intelligence features.

It combines structured task management, AI automation, performance analytics, sprint planning, and real-time collaboration — all within a modern, mobile-first SaaS architecture.

---

# 📌 Project Evolution

TaskPilot AI was developed in three major phases:

* **v1.0 – Core AI Task Manager (MVP)**
* **v2.0 – Collaborative Team Platform**
* **v3.0 – Advanced AI-Powered Execution System**

## ✅ Task Management System

* Create, edit, delete tasks

  * Title

## 🧠 AI Intelligence Layer (Core Differentiator)

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/taskpilot-ai.git
cd taskpilot-ai
```

---

## 2️⃣ Backend Setup

```bash
cd server
npm install
```

Create `.env` file:

```
DATABASE_URL=
JWT_SECRET=
OPENAI_API_KEY=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

Run migrations:

```bash
npx prisma migrate dev
```

Start server:

```bash
npm run dev
```

---

## 3️⃣ Frontend Setup

```bash
cd client
npm install
npm start
```
